Referencia

En esta sección se presentan las páginas web que he desarrollado

Galería Lightbox

Galería de imágenes con efecto Lightbox que permite visualizar las fotos en un tamaño ampliado al hacer clic sobre ellas. Esta página demuestra habilidades en HTML, CSS y JavaScript para crear interactividad y mejorar la experiencia del usuario en la navegación de imágenes

Mi Página Estética

Página web diseñada con un enfoque estético y limpio, mostrando contenido multimedia y texto organizado de forma clara. Esta página resalta el uso de diseño responsivo, estructura de contenido y aplicación de estilos visuales mediante CSS para ofrecer una experiencia atractiva y profesional

Reproductor De Video

El objetivo es mostrar cómo insertar y reproducir videos dentro de una página web sin depender de plataformas externas. El reproductor cuenta con controles para pausar, adelantar o ajustar el volumen, ofreciendo una experiencia completa al usuario.

Animación en CSS

En esta página se presenta una animación creada exclusivamente con código CSS. Se logra movimiento en los elementos de la página. Este ejercicio demuestra el uso del diseño dinámico y la creatividad aplicada al desarrollo web.

Menú De Combinación

Esta página muestra un menú con distintas opciones creadas mediante combinaciones de HTML y CSS. Incluye imágenes que enlazan a otras secciones del sitio, como galerías, formularios y ejemplos de estilos visuales. Su diseño utiliza bordes y fondos de color para mantener una apariencia organizada y atractiva.
